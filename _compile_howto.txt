How to compile FlashFolder
--------------------------

Prerequisites:
- Visual C++ 7.1 (Visual Studio 2003)
- the WiX (Windows Installer XML) toolkit version 2.0 (http://wix.sourceforge.net/)
- environment variable WIX_2_0_DIR set to the directory path of the WiX 2.0 binaries (without trailing backslash)

Compile:
Open FlashFolder.sln in Visual Studio, compile the "Installer" sub-project. This should compile all necessary dependencies resulting in the final .msi installer file.
